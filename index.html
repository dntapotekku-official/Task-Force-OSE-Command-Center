Morning Brief — COO
🔌 API Integration Layer — spesifikasi endpoint internal

Endpoint yang disiapkan untuk backend/gateway internal. Command Center memanggil endpoint ini saat sinkronisasi. Seluruh API key/secret disimpan di environment variable sisi server — frontend hanya menyimpan nama variabelnya, tidak pernah menyimpan key.

  • GET /api/integrations/status — status seluruh connector
  • GET /api/review-rutin/summary — ringkasan eksekutif
  • GET /api/review-rutin/team-summary — capaian per tim & parameter
  • GET /api/review-rutin/outlet-priorities — skor prioritas outlet
  • GET /api/review-rutin/findings — temuan ternormalisasi
  • POST /api/integrations/sync — jalankan sinkronisasi
  • POST /api/findings/:id/create-capa — angkat temuan menjadi CAPA

Struktur data ternormalisasi: source_system, source_record_id, team, outlet_id, review_date, parameter, score, target, status, finding, evidence_url, is_repeated, created_at.

Target dapat diubah langsung pada setiap kartu · status mengikuti target

💾 Ekspor Data (JSON)

      
Unduh seluruh data sebagai file JSON untuk backup atau dibagikan ke manager lain.

      
    

    

      
📥 Impor Data (JSON)

      
Mendukung 2 format: pembaruan temuan (merge — aman, tidak menghapus/menurunkan status) dan backup penuh (mengganti seluruh data).

      
Muat file JSON hasil ekspor. Data saat ini akan diganti.

      
      
    

    

      
📊 Ekspor Excel (CSV)

      
Unduh tabel temuan & CAPA dalam format CSV yang bisa dibuka di Excel.

      
    

    

      
♻️ Muat Ulang dari Cloud

      
Buang salinan lokal di browser ini dan ambil data terbaru dari Cloud Firestore. Berguna bila tampilan tidak sinkron dengan pengguna lain.
